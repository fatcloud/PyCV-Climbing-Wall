## Background subtraction (BS) 

### 說明:
Background subtraction (背景相減)，簡單說是下一個frame減掉現在的frame，一般來說影像相減時會得到一個全黑的畫面，但此時如果物體有移動的話，移動的物件會因為和靜態的背景有明顯的差別，所以可以清楚的看到變化的地方。
